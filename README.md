# rgbjoy.com

Site devlopment for [rgbjoy.com](https://rgbjoy.com)

## Technology used

- [Payload](https://payloadcms.com/)
- Next.js 16
- React 19
- TypeScript
- [R3F](https://docs.pmnd.rs/react-three-fiber/getting-started/introduction) for 3D and shaders
- [Framer Motion](https://www.framer.com/motion/) and [Greensock](https://greensock.com/) for animations
